# Skincare_Silvia
Sistem Manajemen Data Produk Skincare (Katalog)

Deskripsi Project

Sistem Manajemen Data Produk Skincare (Katalog) merupakan aplikasi berbasis web yang dikembangkan untuk mengelola data produk skincare beserta kriteria kandungan produk secara terstruktur dengan memanfaatkan database MySQL.
Project ini dibuat sebagai implementasi mata kuliah Manajemen Basis Data, dengan fokus pada penerapan operasi CRUD (Create, Read, Update, Delete) yang terintegrasi dengan sistem autentikasi admin (login) guna meningkatkan keamanan dan pengelolaan data.

Fitur Utama

Sistem Autentikasi Admin (Secure Login)
Gerbang keamanan utama yang memvalidasi kredensial pengguna terhadap data yang tersimpan di database. Fitur ini memastikan bahwa hak akses pengelolaan produk hanya dimiliki oleh administrator yang sah.

Manajemen Data (CRUD)
Memfasilitasi pengelolaan data produk dan kriteria skincare secara dinamis, meliputi proses penambahan, penampilan, pengubahan, dan penghapusan data.

Integrasi Database
Menggunakan perintah SELECT, INSERT, UPDATE, dan DELETE pada database MySQL untuk memastikan data tersimpan dan ditampilkan secara akurat serta sinkron dengan sistem.
